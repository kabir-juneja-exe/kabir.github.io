HEX Role Creator

Container-based custom HEX color role creator. Users can enter a color code (e.g. 0c0c0c) to receive a unique role.
Requires variable: HEX (value= 0)

Note: To reset a user's access, set their HEX variable back to 0 and remove the role manually.

BDScript 2 
$nomention
$deletecommand
$var[dot;<:dots:1474755436149346510>]
$var[star;]
$addContainer[rolemenu;#000000]
$addTextDisplay[## Auto Role System;rolemenu]
$addSeparator[true;Small;rolemenu]
$addTextDisplay[### $var[star] How to use
$var[dot] Get the HEX colour of your Role.
$var[dot] Click the button below.
$var[dot] Enter the HEX code.
$var[dot] You will get the role!;rolemenu]
$addSeparator[true;Small;rolemenu]
$addActionRow[rolebuttonmenu;rolemenu]
$addButtonCV2[rolebutton;Click Me;success;false;$var[star];rolebuttonmenu]
$addMediaGallery[roleimg;rolemenu]
$addMediaGalleryItem[https://cdn.discordapp.com/attachments/1424703458321367201/1475101167871721542/943c9fa24045aaee2eda79cf43dab92a.gif?ex=699c4284&is=699af104&hm=8ce5517a1ba017eebd137653f4db108bff7d48667dd9090b3e7201db2839c472&;;;roleimg]
$addTextDisplay[-# Enjoy!;rolemenu]
